## gvim設定方法
- ユーザーホームに.vimrc, .gvimrc, .vimを配置
- ユーザーホームの\*rcから.vim内の\*rcを読み込む
```
"~/.vimrc
source ~/.vim/.vimrc
"~/.gvimrc
source ~/.
```
